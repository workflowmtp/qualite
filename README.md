# QC Pilot — MULTIPRINT Unified Multi-Pôles (Next.js)

Application de contrôle qualité industriel convertie en Next.js.

## Installation

```bash
cd qc-pilot-nextjs
npm install
```

## Démarrage

```bash
npm run dev
```

Ouvrir [http://localhost:3000](http://localhost:3000)

## Comptes démo

| Login | Rôle | Mot de passe |
|-------|------|-------------|
| operateur1 | Opérateur | demo |
| chef1 | Chef d'atelier | demo |
| qc1 | Contrôleur QC | demo |
| qualite | Responsable Qualité | demo |
| admin | Admin | demo |
| direction | Direction / DG | demo |

## Stack technique

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS** + CSS custom properties
- **Zustand** (state management avec persistance localStorage)
